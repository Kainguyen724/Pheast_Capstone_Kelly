# Pheast Capstone Project Deliverables - Kelly Nguyen

## Repository Summary
This repository contains the final deliverables for Kelly Nguyen’s Pheast capstone project on CD24 expression in normal pediatric tissues and pediatric tumors. The repository is organized by deliverable type and by project aim so that presentations, reports, Quarto files, code, and poster materials can be located quickly.

## Folder Descriptions

### Powerpoint Presentations
Contains the individual company final Friday Research Meeting presentation summarizing background, aims, datasets, methods, figures, conclusions, and next steps.

### R Quarto Files
Contains the reproducible analysis files and outputs organized by aim.

- **Aim 1 - Healthy Pediatric Peripheral Blood:** Code for the pediatric peripheral blood scRNA-seq analysis, including QC checks, metadata filtering, normalization checks, heatmaps, and expression visualization.
- **Aim 2 - Healthy Pediatric Tissues:** Code for thyroid, thymus, growth plate/bone, and brain analyses. These files include dataset loading, QC, marker-based labeling when needed, and CD24/SIGLEC10 heatmap generation.
- **Aim 3 - KM Analysis for Specific Tumors:** Code for OpenPedCan Kaplan–Meier survival analysis and Cox regression analysis for neuroblastoma, osteosarcoma, and medulloblastoma.
- **BCSM poster:** Code and output related to the poster analysis of CD24 survival trends across all pediatric tumors in OpenPedCan.

### Reports & Documents
Contains written project documentation, including the datasets used document, final individual report, and individual worklog.

### List of Deliverables
Provides the checklist of required general and code deliverables for the SharePoint submission.
